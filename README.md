## Overview

Planetono adalah website 3D bertema makanan dunia dengan pendekatan sinematik dan scroll-driven storytelling. Project ini menampilkan hero 3D burger, food worlds, interactive food atlas, marquee strip, dan library buku kuliner yang dikemas sebagai pengalaman visual premium.

---

## Highlights

- Premium cinematic 3D food website
- Scroll-driven visual storytelling
- 40+ world foods dalam 4 thematic groups
- React Three Fiber hero scene
- Framer Motion transitions
- Interactive world map dan food atlas
- Multi-book reader untuk foods, recipes, dan drinks
